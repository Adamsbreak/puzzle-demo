快捷操作

右键格子：快速添加、移除、清空格子标签。
右键物体：快速改尺寸、颜色、运动规则、复制和删除。
右键区域：快速改尺寸、颜色和位置。
Ctrl + C / Ctrl + V：复制粘贴当前选中的物体或区域。
Delete：删除当前选中的物体或区域。
涂格模式下可以直接跨过物体批量刷格子。
